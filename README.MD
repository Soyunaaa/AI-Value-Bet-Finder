# AI Value Bet Finder

An AI-powered football betting analytics platform.

## Planned Features

- Live Odds Comparison
- Expected Value Calculator
- AI Predictions
- xG Analysis
- Corner Analysis
- Asian Handicap Model
- Kelly Criterion
- Arbitrage Finder
- Historical Odds Tracking

## Stack

Frontend:
- React
- TypeScript
- TailwindCSS

Backend:
- FastAPI
- PostgreSQL

AI:
- Scikit-Learn
- XGBoost
